🚀 Protocole V4

Le Guide de Construction Ultime pour Votre Life OS

💡Notice d'Utilisation Rapide

Suivez ces étapes dans l'ordre pour une construction logique et efficace de votre système.

  1. Construisez d'abord le Noyau Essentiel.

    Créez les 4 bases de données de la Section 1. C'est la fondation.

  2. Mettez en place votre Cockpit.

    Suivez la Section 3 pour construire votre tableau de bord.

  3. Remplissez avec votre Plan d'Action.

    Utilisez la Section 4 pour créer vos premiers projets et tâches.

  4. Intégrez les Modules Optionnels un par un.

    Choisissez le module le plus important pour vous et construisez-le.

  5. Adoptez la Revue Hebdomadaire.

    La Section 6 est la plus importante pour la viabilité à long terme de votre organisation.

🤖 Section 2 : Automatisation & Optimisation

Automatisation 1 : Archiver les Projets

Quand le `Statut` d'un projet passe à `🏁 Terminé`, cochez automatiquement la case `Archiver`.

Automatisation 2 : Synchroniser les Tâches

Quand le `Statut` d'une tâche passe à `✅ Terminé`, cochez automatiquement la case `Fait?`.

Optimisation : Modèles de Projets

Dans votre base `🎯 Projets`, créez un modèle "Nouveau Projet". À l'intérieur de ce modèle, insérez une vue liée de votre base `⚡ Tâches Maîtresse` et filtrez-la pour que `Projet` contienne "Nouveau Projet". Chaque nouveau projet aura ainsi sa propre liste de tâches pré-filtrée.

🕹️ Section 3 : Le Cockpit - Guide de Construction Détaillé

Plan d'Action pour votre Cockpit

Le Cockpit est votre page d'accueil. Il doit vous montrer ce qui est important, maintenant, sans vous submerger. Suivez ces étapes pour le construire.

Étape 1 : La Fondation - La Page "Backend"

Pour garder votre espace de travail propre, nous allons cacher les bases de données maîtresses.

  • Créez une nouvelle page simple et nommez-la `⚙️ Backend`.
  • Déplacez toutes les bases de données que vous avez créées (Tâches, Projets, etc.) à l'intérieur de cette page.
  • Vous n'ouvrirez cette page que rarement. Tout le reste se fera via des vues liées.

Étape 2 : Créer la Page d'Accueil

C'est la page que vous ouvrirez tous les jours.

  • Créez une nouvelle page et nommez-la `🏠 Centre de Commandement` ou `🕹️ Cockpit`.
  • Ajoutez une icône et une image de couverture qui vous inspirent.

Étape 3 : Le Message d'Accueil

Juste en dessous du titre de votre page, créez un encadré pour vous accueillir et vous motiver.

  • Tapez `/encadré` et choisissez "Encadré". Donnez-lui une icône comme `👋`.
  • À l'intérieur, écrivez un message simple. Exemple : "Bonjour ! Prêt à reprendre le contrôle. Quelle est la priorité aujourd'hui ?".

Étape 4 : La Structure en 3 Colonnes

Nous allons maintenant organiser l'espace. Pour créer des colonnes, glissez-déposez un bloc à droite d'un autre.

Étape 5 : Remplir les Colonnes

Dans la Colonne de Gauche :

Créez une seule vue liée de `⚡ Tâches Maîtresse` avec 3 onglets : `🔥 Aujourd'hui`, `📥 Boîte de Réception`, `🗓️ Semaine` (Calendrier). Puis, en dessous, ajoutez un Bouton `➕ Ajouter une tâche`.

Dans la Colonne Centrale :

Créez une vue liée de `🎯 Projets` en affichage Galerie, filtrée pour ne montrer que les projets en cours.

Dans la Colonne de Droite :

Créez un Encadré `🎯 Objectifs de la Semaine` où vous listerez vos 3 priorités. En dessous, créez un titre "Navigation" et glissez-y les liens vers vos pages de Domaines.

Objectif : Avoir une vision claire de mes finances, identifier les fuites et définir des budgets réalistes pour atteindre mes objectifs d'épargne.

Créez une section "Analyse Gemini". Préparez des exports de vos comptes (Caisse d'Épargne, Revolut) sur les 6 derniers mois. Utilisez le prompt suivant :

"Analyse ces transactions. Catégorise chaque dépense (Alimentation, Loyer, Loisirs, Transport, etc.). Identifie les 3 plus grands postes de dépenses variables. Suggère 3 pistes d'économies concrètes et réalistes."

  • Tâche : Soumettre les données à Gemini pour analyse.
  • Tâche : Créer les catégories de budget recommandées dans la base `📊 Budgets`.
  • Tâche : Lister tous mes abonnements dans la base `🔁 Abonnements`.
  • Tâche : Définir des montants pour chaque budget mensuel.
