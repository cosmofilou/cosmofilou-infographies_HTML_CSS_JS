Guide de Construction VBA (Étape par Étape)

Suivez ces étapes dans votre éditeur VBA (Alt+F11) pour recréer la maquette ci-dessus.

Partie 1 : Le "Mode Application" (Code `ThisWorkbook`)

Ce code masque Excel au démarrage et lance votre application (y compris votre `frmSplashScreen`).

  1. Ouvrez l'explorateur de projets (Ctrl+R).
  2. Double-cliquez sur `ThisWorkbook`.
  3. REMPLACEZ TOUT le code existant par celui-ci :
Option Explicit

' --- DÉMARRAGE EN MODE APPLICATION ---
' Cache Excel avant même l'ouverture de la fenêtre principale
Private Sub Auto_Open()
    Application.Visible = False
End Sub

' Lance l'application après le chargement du classeur
Private Sub Workbook_Open()
    ' 1. Affiche le Splash Screen (non bloquant)
    '    (Si vous n'en avez pas, commentez la ligne suivante)
    frmSplashScreen.Show vbModeless
    
    ' Force Excel à traiter les événements d'affichage
    DoEvents

    ' --- Mettez ici vos tâches de démarrage (connexion BD, etc.) ---
    ' Exemple : Call PKS_Golfinho.Button_ServiceOff_Click
    
    ' Pause pour simuler un chargement
    Application.Wait Now + TimeValue("00:00:02")
    ' --- Fin des tâches de démarrage ---

    ' 2. Ferme le Splash Screen
    '    (Si vous n'en avez pas, commentez la ligne suivante)
    Unload frmSplashScreen
    
    ' 3. Affiche l'UserForm principal
    frmMain.Show
End Sub

Partie 2 : Construction Visuelle de `frmMain` (La Coquille)

Ouvrez votre `frmMain` vide en mode visuel (pas le code).

A. Le `UserForm` (`frmMain`)

Sélectionnez le `UserForm` et réglez ses Propriétés :

  • Caption: Golfinho v3.0
  • Height: 720
  • Width: 1000
  • StartUpPosition: 2 - CenterScreen

B. Le Menu (`fraNavigation`)

  1. Prenez un Frame (Cadre) et dessinez-le sur la gauche.
  2. Réglez ses Propriétés :
    • (Name): fraNavigation
    • Caption: (Laissez vide)
    • Top: 0 | Left: 0
    • Height: 720 | Width: 260
    • BackColor: &H111827& (Gris foncé)
    • SpecialEffect: 0 - fmSpecialEffectFlat
    • BorderColor: &H111827&

    C. Le Contenu (`mpContent`)

    1. Prenez un MultiPage (Onglets) et dessinez-le à droite du menu.
    2. Réglez ses Propriétés :
      • (Name): mpContent
      • Top: 0 | Left: 260
      • Height: 720 | Width: 740
      • Style: 2 - fmTabStyleNone (TRÈS IMPORTANT)

      D. Les Pages (dans `mpContent`)

      1. Sélectionnez le mpContent.
      2. Renommez Page1 -> (Name): pageDashboard
      3. Renommez Page2 -> (Name): pagePatrimoine
      4. Faites un clic droit sur "pagePatrimoine" > Nouvelle Page.
      5. Renommez Page3 -> (Name): pageInterventions
      6. Répétez pour : pageReleves, pageFlux, pageAnalyse, pageRapports, pageOutils.

Option Explicit

    Me.mpContent.Value = PAGE_DASHBOARD
    ' Met à jour la couleur du premier bouton de menu
    Call UpdateNavColors(Me.lblNav_Dashboard)

Private Sub UserForm_QueryClose(Cancel As Integer, CloseMode As Integer)
    ' Gère la fermeture de l'application par le 'X'
    
    ' Si l'utilisateur clique sur le 'X'
    If CloseMode = vbFormControlMenu Then
        ' 1. (Optionnel) Pose une question de confirmation

        ' 3. Le UserForm se fermera (car Cancel n'est pas à 1)
        '    et l'utilisateur retrouvera Excel.
    End If
End Sub

' =================================================================
' UTILITAIRE DE NAVIGATION
' =================================================================

Private Sub UpdateNavColors(ByVal ActiveLabel As MSForms.Label)
    ' Réinitialise tous les "boutons" (Labels) du menu
    ' et met en surbrillance celui qui est actif.
    
    Dim ctrl As MSForms.Control
    
    ' 1. Parcourt tous les contrôles DANS le Frame de navigation
    For Each ctrl In Me.fraNavigation.Controls
        ' 2. Vérifie si c'est un Label (Étiquette)
        If TypeName(ctrl) = "Label" Then
            ' 3. Réinitialise sa couleur et son fond
            With ctrl
                .ForeColor = COLOR_MENU_INACTIF
                .BackStyle = fmBackStyleTransparent
                ' (Le BackColor est ignoré si BackStyle est Transparent)
            End With
        End If
    Next ctrl
    
    ' 4. Met en surbrillance le Label actif (Style de la maquette HTML)
    With ActiveLabel
        .BackStyle = fmBackStyleOpaque
        .BackColor = COLOR_FOND_ACTIF
        .ForeColor = &H00FFFFFF ' Blanc
    End With
End Sub

' =================================================================
' UTILITAIRES GÉNÉRAUX
' =================================================================

Private Sub OpenExplorer(ByVal sPath As String)
    ' Ouvre un chemin dans l'Explorateur Windows
    ' Gère les erreurs si le chemin est invalide
    On Error Resume Next
    Shell "explorer.exe """ & sPath & """", vbNormalFocus
    If Err.Number <> 0 Then
        MsgBox "Impossible d'ouvrir le chemin :" & vbCrLf & sPath, vbExclamation, "Erreur de chemin"
    End If
    On Error GoTo 0
End Sub

Partie 5 : Construction des Pages de Contenu (Ex: Dashboard)

Maintenant, remplissez la première page.

  1. Ouvrez `frmMain` en visuel.
  2. Cliquez sur le mpContent et assurez-vous que `pageDashboard` est sélectionnée.
  3. Dessinez-y les `Frame` et les `CommandButton` (Boutons de commande) en suivant la maquette.

A. Le `Frame` "Bienvenue"

  • Propriétés : Caption: (vide), SpecialEffect: 0-Flat, BorderColor: &H00D1D5DB& (Gris clair), BackColor: &H00FFFFFF& (Blanc).
  • Ajoutez un Label dedans pour le texte "Bienvenue...".

B. Le `Frame` "Étape 1: Extraire"

  • Propriétés : Caption: Étape 1: Extraire, Font: Tahoma, Gras, 10.
  • Mêmes styles que "Bienvenue" pour les bordures et le fond.

C. Les Boutons d'Action (ex: `cmdExtractCites`)

  1. Prenez un CommandButton (Bouton de commande).
  2. Dessinez-le dans le `Frame` "Étape 1".
  3. Réglez ses Propriétés :
    • (Name): cmdExtractCites
    • Caption: Extract. Cité(s)
    • Font: Tahoma, Gras, 10
    • BackColor: &H00EB6325& (Bleu)
    • ForeColor: &H00FFFFFF& (Blanc)
    • Height: 40, Width: 160 (par ex.)

    Répétez pour tous les boutons (`cmdExtractClient`, `cmdPRCGEST`, etc.) en ajustant BackColor pour les boutons secondaires (&H0063554B& - Gris foncé).

Partie 6 : Construction de la Page "Outils & Paramètres"

Pour ajouter vos nouveaux boutons :

  1. Ouvrez `frmMain` en visuel.
  2. Cliquez sur le mpContent et sélectionnez l'onglet `pageOutils`.
  3. Prenez un Frame (Cadre) et dessinez-le.
  4. Propriétés : Caption: Accès Outils & Dossiers, Font: Tahoma, Gras, 10.
  5. Prenez un CommandButton et dessinez-le *dans* ce `Frame`.
  6. Réglez ses Propriétés :
    • (Name): cmdOpenJetform
    • Caption: Ouvrir Serveur Jetform
    • BackColor: &H0063554B& (Gris foncé - secondaire)
    • ForeColor: &H00FFFFFF& (Blanc)
    • ... (Même police et taille que les autres boutons)
    1. Copiez ce bouton et réglez les propriétés pour le second :
    • (Name): cmdOpenDemandes
    • Caption: Ouvrir Serveur Demandes
    • ... (Mêmes styles)

    Le code que vous avez copié dans la **Partie 4** s'occupera déjà de gérer les clics sur ces boutons.
